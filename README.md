## Run Locally üíª

```
> Clone the repo
    >> For Windows: Git Bash
    >> For Linux: Terminal
    >> git clone https://github.com/AsishRaju/450-DSA.git
> Change the folder location to your source Code Folder
    >> cd 450-DSA/src
> Install the dependencies
    >> npm install
> Start the server
    >> npm start
> 450DSA runs on port 3000 of your Local Machine
```
